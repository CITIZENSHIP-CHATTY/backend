## CHATTY
Back-end part of chatty chat application

# DEVELOPMENT STUFF

## SYSTEM REQUIREMENTS

* Make
* Docker
* docker-compose

# USAGE

#### Running api service

```bash
make run
```

#### Running api service as daemon

```bash
make run-d
```

#### Build an docker image

```bash
make build
```

#### Removing service containers

```bash
make rm
```

#### Running tests

```bash
make test
```
